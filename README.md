Как получить API-ключ: 
Зайдите на сайт: 
Нажмите Get Free Key — кнопка обычно есть на главной странице.
Заполните форму:
имя;
email;
пароль.
Подтвердите email — откройте письмо от сервиса и перейдите по ссылке.
Войдите в аккаунт (кнопка Login) с теми же данными.
Найдите API‑ключ в личном кабинете (раздел API Keys или Dashboard).
Скопируйте ключ — это длинная строка букв и цифр.
примеры использования:
подтверждает вашу личность (аутентификация);
даёт нужные права (авторизация);
помогает считать запросы (учёт лимитов);
защищает сервис от неавторизованного доступа.
