# Currency Converter (Конвертер валют)
Автор: Василиса Матвеева  
Описание: Графическое приложение для конвертации валют с использованием актуальных курсов с API, которое:  
- Получает актуальные курсы валют с сайта exchangerate-api.com  
- Сохраняет историю конвертаций  
- Предоставляет интуитивно понятный интерфейс для конвертации  
Возможности:  
- Поддержка множества валют (USD, EUR, GBP, JPY и др.)  
- Данные об актуальных курсах валют  
- Сохранение и загрузка истории конвертаций  
- Проверка ввода на положительные значения сумм  
Инструкция:  
1. Перейдите на сайт [exchangerate-api.com](https://www.exchangerate-api.com/)  
2. Нажмите "Get Free API Key"  
3. Зарегистрируйтесь, указав email  
4. Получите ключ в личном кабинете (пример ключа: `1234567890abcdef`)  
5. Вставьте полученный ключ в код программы вместо `YOUR_API_KEY`
Пример работы приложения:
1. Введите сумму: `100`  
2. Выберите валюту из: `USD`  
3. Выберите валюту в: `EUR`  
4. Нажмите "Convert"  
5. Ожидаемый результат: Отображение суммы в евро по текущему курсу
