# Random-Password-Generator
Автор: Василиса Матвеева  
Описание программы: генератор случайных паролей — это программа, которая автоматически создаёт уникальные комбинации символов по заданным пользователем правилам. История паролей сохраняется в формате JSON. Интерфейс интуитивно понятен и прост в использовании. Ограничение длины пароля от 4 до 32 символов. Таблица истории отображает последние 20 сгенерированных паролей. Подробное описание параметров генерации для каждого пароля в истории. Приложение готово к использованию и дальнейшему развитию (например, можно добавить копирование пароля в буфер обмена или экспорт истории в файл).  
Инструкция:  
1. Генерация пароля: установите длину пароля - перетащите ползунок от 4 до 32 символов  
Выберите типы символов - отметьте чекбоксы:  
✓ Заглавные буквы  
✓ Строчные буквы  
✓ Цифры  
Специальные символы  
Нажмите "Сгенерировать пароль"  
Скопируйте пароль - нажав на кнопку копирования рядом с сгенерированным паролем  
2. Работа с историей  
Просмотр истории - все сгенерированные пароли отображаются в таблице  
Фильтрация - используйте чекбокс "Показать только текущие настройки" для фильтрации  
Поиск по дате - используйте поисковик таблицы для поиска по дате  
Очистка истории - кнопка "Очистить историю" удаляет все записи  
3. Сохранение данных  
История автоматически сохраняется в файл password_history.json
